Online Salon Provider 💇‍♀️✨
A modern full-stack salon appointment booking platform where users can explore salon services, book appointments, manage schedules, and enjoy a seamless salon experience online.
🚀 Live Demo
🌐 Live Website: https://salon-appointment-two.vercel.app
📌 Features
🔐 User Authentication & Authorization
📅 Online Appointment Booking
💇 Browse Salon Services
👩‍🎨 Stylist/Salon Management
📱 Fully Responsive UI
⚡ Fast and Smooth User Experience
🛠 Admin Dashboard Support
☁️ Cloud Deployment with Vercel

⚙️ Installation & Setup
1️⃣ Clone Repository
Bash
git clone https://github.com/Shivam9572/online-salon-provider.git
2️⃣ Move into Project Folder
Bash
cd online-salon-provider
3️⃣ Install Dependencies
Bash
npm install
4️⃣ Start Frontend
Bash
cd client
npm start
5️⃣ Start Backend
Bash
cd server
npm start
🌍 Environment Variables
Create a .env file in the server directory and add:
Environment
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
